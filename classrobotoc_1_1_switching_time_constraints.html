Constructor.

Parameters
[in]max_num_switchesMaximum number of switches on the horizon.
[in]min_dtMinimum dwell-time. Default is std::sqrt(std::numeric_limits<double>::epsilon()).
[in]min_dt0Minimum time margin between the initial time of the horizon and initial switching time. Default is std::sqrt(std::numeric_limits<double>::epsilon()).
[in]min_dtfMinimum time margin between the terminal time of the horizon and last switching time. Default is std::sqrt(std::numeric_limits<double>::epsilon()).
[in]barrierBarrier parameter. Must be positive. Should be small. Default is 1.0e-04.
[in]fraction_to_boundary_ruleParameter of the fraction-to-boundary-rule Must be larger than 0 and smaller than 1. Should be between 0.9 and 0.995. Default is 0.995.
