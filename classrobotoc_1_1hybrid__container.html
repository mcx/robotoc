Detailed Description

template<typename Type, typename ImpulseType = internal::EmptyType, typename SwitchingType = internal::EmptyType>
class robotoc::hybrid_container< Type, ImpulseType, SwitchingType >

A container used for formulating the hybrid optimal control problem. This container has data for time stages (with Type), data for lift stages (with Type), data for aux stages (with Type), data for impulse stages (with ImpulseType), and data for switching ingredients.

Template Parameters
TypeThe type name of the standard data type.
ImpulseTypeThe type name of the impulse data type. Defalt is internal::EmptyType.
SwitchingTypeThe type name of the switching data type. Defalt is internal::EmptyType.
